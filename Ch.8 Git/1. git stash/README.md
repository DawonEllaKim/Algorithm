# 1. git stash

<aside>
❗ git stash는 변경사항을 commit 없이 임시로 저장할 수 있도록 도와주는 기능

</aside>

# `git stash`

- commit 없이 변경 사항을 임시저장한다.

# `git stash list`

- 지금까지 임시저장 해 놓은 목록들을 보여준다.

# `git stash show stash@<index>`

- 특정 stash의 로그를 보여준다.

# `git stash pop`

- 제일 마지막에 임시저장한 사항을 불러 온다.
