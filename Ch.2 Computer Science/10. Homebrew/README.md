# 10. Homebrew

## 🔖 홈브루

- 맥OS용 패키지 관리자
- 쉽게 패키지를 관리 (설치 + 삭제)할 수 있다.

## 🔖 홈브루 설치

```jsx
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

```jsx
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```
